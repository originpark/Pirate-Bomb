# Pirate Bomb

## 介绍

2D横板像素游戏

使用`Godot`游戏引擎: [Godot Engine - Free and open source 2D and 3D game engine](https://godotengine.org/)

美术资源: [Pirate Bomb by Pixel Frog](https://pixelfrog-assets.itch.io/pirate-bomb)
